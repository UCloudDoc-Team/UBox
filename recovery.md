# 自动容灾

两个盒子互联组成一对，实际网络链路通过主盒子联通。
当主盒子发生故障（如硬件面卡死、断网等情况），网络链路会自动切换至备用盒子。



![图片3](images/图片3.png)



**主备切换时间实测如下：**

| **VPN智能网关** | **公网** | **云主机** |
| --------------- | -------- | ---------- |
| 3秒             | 4秒      | 4秒        |
| 2秒             | 3秒      | 3秒        |
| 3秒             | 4秒      | 4秒        |
